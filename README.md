A lightweight **ARP-based network scanner** built with **Python and Scapy** for **authorized network discovery and cybersecurity learning**.  
The tool scans a given **CIDR range** on a local network and identifies live hosts along with their IP address, MAC address, and hostname.

---

##  Features
- ARP scanning for accurate local network host discovery
- Multithreaded scanning for faster performance
- Retrieves IP address, MAC address, and hostname
- Simple and clean terminal output
- Beginner-friendly Python code

- Queue
